PadelPancakes

Tools/languages: HTML, CSS, JS, SASS, Photoshop and Adobe XD

As one of my favorite hobbies is playing a game of padel with my friends, developing a website for our team was a great opportunity for me to start building my front-end skills.

When browsing through the website, you can find the team ranking, the overview of the courts that are booked for the upcoming period and the overview of sets ended with a bagel, a 6-0 set. As it is important that the users can access the booked courts and other data at any moment, the website is mainly intended for use through the smartphone and therefore developed based on the Mobile-First approach.

Similarly as with my portfolio website, I leveraged a broad range of my front-end and UI/UX skills. I have designed the website with Adobe XD and Photoshop, and developed it with HTML, CSS, JS and SASS.

For the full experience, visit www.padelpancakes.be.

Javascript Projects

Tools/languages: HTML, CSS and JS

I have completed a number of projects using my accumulated knowledge about JS, including a task list, number guesser, loan calculator, etc.

One example that I would like to highlight is a task list. This application allows you to insert a new task which will be added to the list below. When tasks are added to the list, you can delete items from the list or clear the whole list. When a user fills in a task without a value, a message will appear prompting the user to fill in a task. For this project, localStorage is utilized to remember the task list. This way, you can continue working on the task list when the application is reopened.

Maynooth

Tools: Adobe XD and Photoshop

During a course about UX design essentials, I have designed a webshop for “Maynooth”, a furniture store. The project entailed the development of a mockup, which was created with a given description of a persona in mind. Most of the work was performed in Adobe XD, in combination with Photoshop for the editing of images.

Specifically for this website, I have created custom icons with Adobe XD, which were implemented in the mockup. To make this mockup interactive, prototypes or wires were utilized and micro interactions were used to create interactivity on the page itself.

During this project, I also got accustomed to the UX process and the different steps, including the user testing.
